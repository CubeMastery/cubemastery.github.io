\/ rules and info \/

This year CubeCTF will be an Attack/Defense CTF themed around human-in-the-loop. Vulnbox access is through a web console, with package mirrors available for installing tools, and all other internet access blocked. Agentic AI use is banned (you may ask AI questions, but may not run agents). Teams found violating the AI policy may be disqualified and publicly shamed.

Range infrastructure is hosted by Glitch Range. Since only terminal access is provided, we recommend brushing up on vim, tcpdump and mitmproxy. See the game format docs for environment info

The goals of this event are to run a public beginner-friendly Attack/Defense CTF (while keeping all skill levels engaged), and to experiment with mitigations against CTF slopping. If those goals sound fun, register on Discord!

  • No denial of service
